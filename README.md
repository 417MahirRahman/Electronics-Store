# üìñ About

TechNest is a beginner-friendly mini e-commerce web application built using Next.js (App Router). The project focuses on building a clean and responsive product browsing experience while learning modern frontend concepts such as routing, component structure, and data handling.

The application displays a list of electronic products sourced from a local JSON data file, allowing users to browse items and view detailed product information without relying on a backend API. This approach keeps the project simple while still reflecting real-world data-driven UI patterns.

A basic mock authentication system is implemented using hardcoded credentials and cookies to simulate login behavior and protect specific routes, such as the product creation page. Public pages remain accessible to all users, while protected pages demonstrate basic access control logic.

The user interface is styled with Tailwind CSS and DaisyUI, enabling rapid UI development with consistent and reusable components. TechNest is designed as a learning-focused project that helps strengthen understanding of Next.js App Router, route protection, and structured frontend development.

# ‚ú® Main Features

###
1.Product Listing from Local JSON Data  
Displays a collection of electronic products loaded from a structured JSON file, simulating real-world data-driven UI without a backend API.

2.Dynamic Product Details Pages  
Implements dynamic routing to show detailed information for each product, including description, price, and images.

3.Mock Authentication with Route Protection  
Uses hardcoded credentials and cookies to simulate authentication and restrict access to protected pages.

4.Protected Add Product Page  
Allows only authenticated users to add new products, demonstrating basic access control and form handling.

5.Modern Responsive UI with DaisyUI  
Built with Tailwind CSS and DaisyUI to deliver a clean, responsive, and consistent user interface across devices.
